# 🚀 Mini Web Projects Collection

This repository contains a collection of small web-based projects built using HTML, CSS, and JavaScript. These projects are created to practice core web development concepts and improve problem-solving skills.

---

## 📌 Projects Included

### 📅 Calendar App

* Simple and interactive calendar UI
* Displays current date, month, and year
* User-friendly design

### 🧮 Calculator

* Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
* Clean and responsive UI
* Keyboard input support (optional if added)

### 🎤 Voice Converter

* Converts text to speech / speech to text (depending on your feature)
* Uses browser APIs for voice processing
* Easy to use interface

### 🍔 Food App (Single Page Website)

* Attractive food ordering UI
* Fully responsive single page design
* Sections like Home, Menu, About, Contact
* Smooth scrolling and modern layout
